Input

Output

Here for output we use Output parameter and event Emitter. Then we have to use the event emitter parameter and have to assign value of a function of the parent ts file where is finds the matched value. Output is coming from Chiled from emitting and we are passing that emitted value or object to the function of parent component like (selectedPatientId)="handleChildResponse($event). selectedPatientId is Output variable in child component which is emitting the clicked patient which is another event called on click of a patient and emit the clicked one. handleChildResponse is a function in parent that is taking the emitted value
